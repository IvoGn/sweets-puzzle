# Sweets Puzzle

Sweets Puzzle ist ein Match-3 Puzzle-Spiel, inspiriert von Klassikern wie Candy Crush.
Das Projekt wurde im Rahmen einer Webentwicklungsausbildung umgesetzt und legt den Fokus auf saubere Struktur, UI-Interaktion und Spielmechaniken.

---

## Features

* **Match-3 Gameplay**
  Tausche benachbarte Süßigkeiten, um Kombinationen zu bilden und Punkte zu sammeln

* **Score & Highscore System**
  Verfolge deinen aktuellen Punktestand und den besten Score der Session

* **In-Game Währung**
  Verdiene Geld durch Kombinationen und nutze es im Shop

* **Shop-System**
  Kaufe Boni zur Unterstützung im Spiel

* **Boni-System**

  * Extrarunde
  * Tausch nicht benachbarter Süßigkeiten
  * Umwandlung einer Süßigkeit

* **Rundenbasiertes Spielsystem**
  Jeder Zug verbraucht eine Runde

* **Pause-Funktion**
  Spiel kann jederzeit pausiert werden

* **Sound & Animationen**

  * Hintergrundmusik
  * Soundeffekte für Aktionen
  * Animationen beim Tauschen und Entfernen
